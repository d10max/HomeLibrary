Library Management System (Console)

Технологічний стек
Мова: C# (.NET 8.0)
ORM: Entity Framework Core
База даних: SQLite
Архітектура: Console Application з розділенням логіки (Models, Data, Services).

Додаток підтримує повний цикл CRUD-операцій:
Додавання книг: Якщо автор вже існує в базі — книга прив'язується до нього; якщо ні — створюється новий запис автора.
Перегляд: Вивід списку всіх книг з підтягуванням даних про авторів.
Пошук: Реалізований за допомогою Алгоритму Левенштайна.
Редагування/Видалення: Можливість змінювати інформацію про книгу або видаляти її з бази.
